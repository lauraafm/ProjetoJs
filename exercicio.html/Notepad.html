O que é o notepad++?

Notepad++ é um editor de texto e de código fonte de código aberto sob a licença GPL. Suporta várias linguagens de programação rodando sob o sistema Microsoft Windows. O Notepad++ é distribuído como um Software livre. Ideal para desenvolvedores, front e back-end
